# 東京高専社会実装第9班の廃棄野菜有効活用システム（開発中）
### 概要
廃棄野菜の取引サイト。
廃棄野菜を減少させ、必要な人のところへいきわたらせることを目的としている。

### 使用ツール
django 

### 使用言語
javascript,HTML,python,CSS
